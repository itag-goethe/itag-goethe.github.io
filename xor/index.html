Aufgabe XOR: Testfälle

Auf dieser Seite stehen fünf Dateien mit Testfällen bzw. Beispieleingaben zur Aufgabe 11 (XOR) bereit. Jede Datei enthält in 50 Zeilen jeweils ein Testfall pro Zeile bestehend aus einer positiven ganzen Zahl (der Anzahl der Server im Rechenzentrum). Als Ausgabe ist eine Datei mit 50 Zeilen und den entsprechenden minimalen Kosten für jeden Testfall zu liefern. Die Eingabegrößen und Dateien sind folgender Tabelle dargestellt.

Beschränkungen n <= 5 n <= 25 n <= 3000 n <= 10^5 n <= 10^12

Zu einer vollständigen Lösung gehören eine Beschreibung des Algorithmus, der Quelltext des zugehörigen Programms und die Lösungen bzw. korrekten Ausgabedateien zu den obigen Eingabedateien. Wer die gesamte Aufgabe (Teil (i), Teil (ii) sowie Teil (iii)) als erstes allein oder im Team selbstständig korrekt löst, erhält einen Raspberry PI Einplatinencomputer.
